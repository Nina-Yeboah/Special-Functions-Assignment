# Special-Functions-Assignment
##CONTENTS
Special Functions      ---------------------------------------------------------------------          Description
1.constructor __init__()                                                                     Initializes a new object attribute
2. string representation __str__(), __repr__()     Returns a clean readable text(str), Debugging in interactive console(repr)  
3. length __len__()                                                                   Returns integer showing the size or count
4. equality __eq__()                                                               Defines how two objects are considered equal
5. operator overloading ___add()__ , __sub__(), __mul()        Overloads the addition, subtraction and multiplication operators
6. item access __getitem__()                                                         Allows reading data using bracket notation
7. item assignment __setitem__()                                                   Allows assigning data using bracket notation
8. membership testing __contains__()                                                                  Returns 'True' or 'False'
9. callable objects __call__()                                                 Makes an object callable like a regular function
10.iteration support __iter__(), __next__()                                                          Returns an iterator object
11.Boolean evaluation __bool__()                                                                  Defines how true an object is
12.Destructor __del__()                              Cleans up resources right before object is destroyed and garbage-collected
